# Getting Started

This is a GNSS Software Defined Radio (SDR) implemented in python, based on SoftGNSS 3.0 developed by Darius Plausinaitis and Dennis M. Akos in Matlab.

# System requirements

* python 2.7/3
* matplotlib
* scipy
* numpy
